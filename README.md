# 基于Vue的移动端旅游网应用
### 技术栈
    vue+vue-router+vuex
### 主要实现
    1. 使用 rem 适配不同的手机屏幕
    2. 使用 less 编写页面的样式
    3. 使用 vue-router 编写路由，使用 编程式导航 实现路由间的跳转
    4. 使用 axios 请求数据(数据为本地mock的数据，存放在public/static/mock)
    5. 将公共组件进行拆分
    6. 使用 vuex 实现多页面共享数据
    7. 使用 函数节流 实现列表性能优化 
    8. 使用 keep-alive 优化网页性能，减少不必要的请求 
    9. 使用 slot 插槽实现简单的动画封装
